Ei saa aru, kuidas peaks neid väärtusi määrama koodis. Praegu olen aadressi vaadanud inimloetavalt lehelt https://register.muinas.ee/rest/v1/browser.php Võtan kõikide mälestiste data. Määrata tuleb limit, et võtaks kõik asjad, mitte default ainult 20.

res = GET("https://register.muinas.ee/rest/v1/monuments?",
          query = list(
              limit = 100
          ))
